A simple 3D horror/spooky game based on a nosleep story of WorldAwayTweedy I saw on Reddit. I find the idea of that story is very interesting so I decided to make a simple game based on it.
The game featured a dialogue system, simple character movement and interaction, minigames.
This game will not have a save system because it is a short game and I want players to go through the entire story in only 1 playthrough.
I will publish this game onto Itch.io once it is playable and finished.
